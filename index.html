Wireframe

Wireframe é uma representação visual da estrutura básica de uma página. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou scaneie o QR Code para obter o wireframe desta página
